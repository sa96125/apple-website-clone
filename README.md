## 실행 방법

1. **환경 설정:**

   - `npm install`을 사용하여 필요한 패키지를 설치합니다.

2. **개발 서버 실행:**

   - `npm run dev` 명령어로 개발 서버를 실행합니다.

3. **빌드 실행:**

   - `npm run build` 명령어로 정적 파일을 생성합니다.
